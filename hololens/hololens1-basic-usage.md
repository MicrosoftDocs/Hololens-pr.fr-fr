# <a name="getting-around-hololens-1st-gen"></a>Se familiariser avec HoloLens (1ère génération)

Vous êtes prêt à parcourir le monde des hologrammes ? Voici quelques informations sur la prise en main.

Ce guide fournit une introduction à la réalité mixte, aux gestes pour l’interaction avec les hologrammes, et à une introduction à Windows holographique.

## <a name="discover-mixed-reality"></a>Détection de la réalité mixte

Sur HoloLens, les hologrammes se mélangent avec votre environnement physique pour regarder et paraître comme dans votre monde. Même lorsque les hologrammes vous concernent, vous pouvez toujours voir votre environnement, vous déplacer librement et interagir avec d’autres personnes et objets. Nous appelons cette expérience « réalité mixte ».

Le cadre holographique positionne vos hologrammes là où vos yeux sont les plus sensibles aux détails et les lentilles de visualisation laissent la vision de votre périphérique non cachée. Avec un son spatial, vous pouvez identifier un hologramme même s’il se trouve derrière vous. Étant donné que HoloLens apprend et comprend votre environnement, vous pouvez placer des hologrammes sur des objets réels et, par conséquent, faire de vos applications et de vos jeux. Par conséquent, un personnage dans un jeu peut s’asseoir sur votre canapé, ou [des robots d’espace peuvent sortir de vos murs](https://www.microsoft.com/store/apps/9nblggh5fv3j).

## <a name="use-hololens-with-your-hands"></a>Utilisez HoloLens avec vos mains

La prise en main de HoloLens est beaucoup plus semblable à l’utilisation de votre smartphone. Vous pouvez utiliser vos mains pour manipuler des fenêtres, des menus et des boutons holographiques.  Au lieu de pointer, de cliquer ou de toucher, vous allez utiliser votre point de regard, votre [voix](hololens-cortana.md)et vos gestes pour sélectionner des applications et des hologrammes et pour vous familiariser avec HoloLens.

Lorsque vous connaissez ces interactions de base, l’approche de HoloLens est un jeu d’enfant.

Nous vous guiderons à travers les bases la première fois que vous utilisez votre HoloLens. Vous trouverez également un didacticiel sur les gestes dans votre menu **Démarrer** : recherchez l’application de mouvements d’apprentissage.

### <a name="the-hand-tracking-frame"></a>Frame de suivi de la main

HoloLens possède des capteurs qui peuvent voir quelques centimètres de l’un des côtés de vous. Lorsque vous utilisez vos mains, vous devez les conserver à l’intérieur de ce frame, ou HoloLens ne les verra pas. Lorsque vous vous déplacez, le frame se déplace avec vous.  

![Image qui montre le frame de suivi HoloLens](./images/hololens-2-gesture-frame.png)

### <a name="open-the-start-menu-with-bloom"></a>Ouvrir le menu Démarrer avec des fleurs

Pour ouvrir le menu **Démarrer** :

1. Maintenez votre main devant vous dans le cadre du geste.
1. Fleur : Mettez tous vos doigts en ensemble, puis ouvrez votre main.
  ![Animation qui montre le mouvement de floraison](./images/hololens-bloom.gif)

### <a name="select-holograms-with-gaze-and-air-tap"></a>Sélectionner des hologrammes avec le robinet et le toucher à l’air

Pour sélectionner une application ou un autre hologramme, appuyez sur celle-ci tout en regardant directement l’hologramme que vous sélectionnez. Pour ce faire, procédez comme suit :

1. Pointez avec le regard de l’hologramme que vous souhaitez sélectionner.
1. Pointez votre index vers le haut pour atteindre le plafond.
1. TAP Air : Abaissez votre doigt, puis augmentez-le rapidement.
   ![Animation de mouvement d’appui sur l’air](./images/hololens-air-tap.gif)

### <a name="select-a-hologram-by-using-your-voice"></a>Sélectionner un hologramme à l’aide de votre voix

1. Le point de regard est un point que vous déplacez en déplaçant votre tête. Vous pouvez l’utiliser pour cibler des commandes vocales avec précision.
1. Pointez avec le regard de l’hologramme que vous souhaitez sélectionner.
1. Pour sélectionner l’hologramme, dites « sélectionner ».

## <a name="holograms-and-apps"></a>Hologrammes et applications

À présent, il est temps de placer les gestes dans le test !

Vous trouverez vos applications installées dans le [menu Démarrer](holographic-home.md) et d’autres applications pour HoloLens (1re génération) dans le Microsoft Store.

Ouvrez le menu **Démarrer** et sélectionnez une application !

L’utilisation d’applications sur HoloLens est légèrement différente de celle d’un PC : certaines applications utilisent une vue 2D et ressemblent à d’autres applications Windows. D’autres applications (applications immersives) utilisent une vue 3D et, lorsque vous les lancez, elles deviennent la seule application que vous voyez.

Quand vous placez une fenêtre d’application ou un lanceur d’applications, celle-ci reste en place jusqu’à ce que vous la supprimiez. Vous pouvez déplacer ou redimensionner ces hologrammes dans votre foyer de réalité mixte à tout moment.

## <a name="move-resize-and-rotate-apps"></a>Déplacer, redimensionner et faire pivoter des applications

Le déplacement et le redimensionnement d’applications sur HoloLens fonctionnent un peu différemment de sur un PC. Au lieu de faire glisser l’application, vous allez utiliser votre point de regard, ainsi qu’un [geste](https://support.microsoft.com/help/12644/hololens-use-gestures) ou un [clic](hololens1-clicker.md). Vous pouvez également faire pivoter une fenêtre d’application dans l’espace 3D.

> [!TIP]
> Réorganisez les applications à l’aide de votre voix-pointez sur une application et dites « Je suis », « plus grande » ou « plus petite ». Ou que Cortana déplace une application pour vous : dites « Hey Cortana, déplacer le \* *nom \** de l’application ici ».

### <a name="move-an-app"></a>Déplacer une application

Pointez avec le regard de l’application (dans la barre de titre d’une fenêtre d’application), puis effectuez l’une des opérations suivantes.

- Appuyez et maintenez la touche enfoncée pour sélectionner l’application. Déplacez votre main pour positionner l’application et augmentez votre doigt pour la placer.
- Sélectionnez **ajuster**, maintenir le stylet et déplacer votre main pour positionner l’application. Augmentez votre doigt pour la placer, puis sélectionnez **terminé**.
- Sélectionnez **ajuster**, cliquez et maintenez le curseur enfoncé et déplacez votre main pour positionner l’application. Libérez le clic, puis sélectionnez **terminé**.

> [!TIP]
> Si vous déposez des applications lorsque vous les déplacez, veillez à garder votre main dans le cadre de mouvement en le suivant avec votre point de regard.

### <a name="resize-an-app"></a>Redimensionner une application

Pointez avec le regard de l’application, puis effectuez l’une des opérations suivantes.

- Placez le regard sur un coin ou un bord d’une fenêtre d’application, puis appuyez et maintenez enfoncé. Déplacez votre main pour modifier la taille de l’application et augmentez votre doigt quand vous avez terminé.
- Sélectionnez **ajuster**. Placez le regard sur l’un des carrés bleus aux angles de l’application, appuyez et maintenez, puis déplacez votre main pour redimensionner l’application. Augmentez votre doigt pour le libérer, puis sélectionnez **terminé**.
- Sélectionnez **ajuster**. Placez le regard sur l’un des carrés bleus aux angles de l’application, cliquez et maintenez enfoncé le clic, puis déplacez votre main pour redimensionner l’application. Libérez le clic, puis sélectionnez **terminé**.

> [!TIP]
> En mode d’ajustement, vous pouvez déplacer ou redimensionner n’importe quel hologramme.

### <a name="rotate-an-app"></a>Faire pivoter une application

Pointez avec le regard de l’application, puis appuyez et maintenez les deux mains pour la sélectionner. Faites pivoter l’application en gardant une main stable et en déplaçant votre autre main. Lorsque vous avez terminé, augmentez les deux doigts d’index.

### <a name="scroll-content-in-an-app-window"></a>Faire défiler le contenu dans une fenêtre d’application

Pointez avec le regard du contenu de la fenêtre d’application. Appuyez et maintenez, puis déplacez votre main légèrement vers le haut ou vers le bas pour faire défiler le contenu.

### <a name="help-i-dont-see-my-holograms"></a>Au secours ! Je ne vois pas mes hologrammes

Si vous ne voyez pas les hologrammes que vous avez placés lors de l’utilisation de HoloLens, voici quelques points à essayer :

- Assurez-vous que vous regardez dans la zone de droite, mais les &mdash; hologrammes restent là où vous les avez laissés !
- Assurez-vous que vous êtes dans une salle bien éclairée sans grand soleil.
- Qu'. Lorsque HoloLens rencontre des difficultés pour reconnaître votre espace, les hologrammes précédemment placés peuvent prendre jusqu’à une minute pour réafficher.
- Si le problème persiste, vous pouvez effacer vos données de stockage hologrammes dans **paramètres**  >  **système**  >  **hologrammes**, puis placer à nouveau des hologrammes dans une page d’hébergement de la réalité mixte.
