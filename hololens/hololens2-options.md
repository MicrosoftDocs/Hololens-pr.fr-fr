# <a name="hololens-2-overview"></a>Vue d’ensemble de HoloLens 2

### **<a name="hololens-2-industrial-edition"></a>HoloLens2 Industrial Edition**

#### <a name="learn-about-hololens-2-industrial-edition"></a>En savoir plus sur HoloLens2 Industrial Edition
Nous avons amélioré, puis repensé HoloLens 2 Industrial Edition avec de nouveaux matériaux et certifications pour prendre en charge un ensemble plus vaste d’environnements industriels, notamment les suivants:

- **Environnements de salle propre:** HoloLens 2 Industrial Edition respecte les conditions établies pour la classe ISO 5 de la norme ISO 14644-1 relative aux salles propres où la quantité de particules étrangères est limitée et restreinte, et les objets de ces espaces ont des limites sur l'excrétion, le dégazage et la rétention des particules.
- **Emplacements dangereux :** HoloLens 2 Industrial Edition prend en charge les environnements comportant des gaz, des vapeurs ou des liquides inflammables, mais où ces substances ne sont généralement pas présentes dans des concentrations inflammables pendant des opérations normales. Il bénéficie d’une certification UL de classe I, division 2, groupes A, B, C et D HAZLOC.

En plus de l’appareil mis à niveau, HoloLens 2 Industrial Edition inclut une garantie de 2 ans et un programme de remplacement rapide.

#### <a name="redeem-your-credits"></a>Échangez vos crédits

##### <a name="unity"></a>Unity
Vous aurez reçu un code unique. Instructions d’échange de la version d’essai Unity Pro :
1. Veuillez consulter le site [id.unity.com](http://id.unity.com/)
1. Connectez-vous avec votre ID Unity ou créez-en un
1. Cliquez sur **Échanger un code de produit**
1. Entrez le code du produit, puis cliquez sur **Échanger un code de produit**

Si vous souhaitez en savoir plus, veuillez consulter la page [sur l’activation d’une licence Unity.](https://support.unity3d.com/hc/articles/211438683-How-do-I-activate-my-license-) Veuillez contacter [le support Unity pour](https://support.unity3d.com/hc) obtenir de l’aide si nécessaire.  

##### <a name="pixyz"></a>Pixyz
Vous aurez reçu un code unique. Instructions d’échange de la version d’essai du plug-in Pixyz :
1. Veuillez consulter le site [id.unity.com](http://id.unity.com/)
1. Cliquez sur **Échanger un code de produit**
1. Entrez votre code de produit du plug-in Pixyz, puis cliquez sur **Échanger un code de produit**

Si vous souhaitez en savoir plus, veuillez consulter la page sur [l’échange de votre licence Pixyz.](https://www.pixyz-software.com/documentations/html/2020.1/review/TrialLicense.html) Veuillez contacter [le support Pixyz pour](https://www.pixyz-software.com/support/) obtenir de l’aide si nécessaire.
